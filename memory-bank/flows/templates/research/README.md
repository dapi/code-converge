# Research Templates Index

Начни с package `README.md` и `brief.md`. Добавляй `plan.md` только по trigger из Research & Discovery Flow; `evidence.md`, `synthesis.md` и `decision.md` появляются по мере перехода lifecycle.

- [`package-README.md`](package-README.md) — package index со ссылкой на lifecycle owner.
- [`brief.md`](brief.md) — canonical decision question, scope and lifecycle owner.
- [`plan.md`](plan.md) — conditional research-method owner.
- [`evidence.md`](evidence.md) — provenance and observation log.
- [`synthesis.md`](synthesis.md) — findings, confidence and limitations.
- [`decision.md`](decision.md) — decision rationale and promotion map; terminal state остаётся в `brief.md`.
